## 1. Qdrant简介

Qdrant是一个高性能的向量相似度搜索引擎和向量数据库，专为AI应用、机器学习和现代搜索引擎设计。它提供了高效的向量存储、检索和过滤能力。

### 1.1 核心特性

| 特性 | 描述 |
|------|------|
| **高性能** | 使用Rust编写，内存安全且高效 |
| **过滤搜索** | 支持向量搜索与payload过滤结合 |
| **量化支持** | 支持Scalar、Product、Binary量化 |
| **分布式** | 支持水平扩展和分片 |
| **持久化** | 支持磁盘存储和快照 |
| **多语言SDK** | 支持Rust、Python、Go、Java等 |

### 1.2 版本信息

- **当前稳定版本**: v1.7+
- **Rust客户端**: qdrant-client crate
- **通信协议**: gRPC (端口6334) + REST API (端口6333)

## 2. 核心功能

### 2.1 向量存储与检索

```
┌─────────────────────────────────────────────────────────────┐
│                     Collection (集合)                        │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    Point (点)                         │   │
│  │  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  │   │
│  │  │     ID      │  │   Vector    │  │   Payload   │  │   │
│  │  │  (u64/UUID) │  │  (f32[])    │  │  (JSON)     │  │   │
│  │  └─────────────┘  └─────────────┘  └─────────────┘  │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 支持的距离度量

| 度量类型 | 公式 | 适用场景 |
|---------|------|---------|
| **Cosine** | 1 - (A·B)/(‖A‖‖B‖) | 文本嵌入、语义搜索 |
| **Euclidean** | √Σ(Aᵢ-Bᵢ)² | 图像特征、坐标 |
| **Dot Product** | A·B | 归一化向量、推荐系统 |

### 2.3 索引类型

#### HNSW (Hierarchical Navigable Small World)

```
Layer 2:     ●───────────────●
              │               │
Layer 1:     ●───────●───────●───────●
              │       │       │       │
Layer 0:     ●───●───●───●───●───●───●───●
```

**HNSW配置参数**:

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `m` | 16 | 每层最大连接数 |
| `ef_construct` | 100 | 构建时的搜索范围 |
| `full_scan_threshold` | 10000 | 全扫描阈值 |

### 2.4 量化支持

| 量化类型 | 压缩比 | 精度损失 | 适用场景 |
|---------|--------|---------|---------|
| **Scalar** | 4x | 低 | 通用场景 |
| **Product** | 8x-64x | 中 | 大规模数据 |
| **Binary** | 32x | 高 | 快速检索 |

### 2.5 Payload过滤

```rust
// 支持的过滤条件类型
Filter::must([...])      // AND 条件
Filter::should([...])    // OR 条件
Filter::must_not([...])  // NOT 条件

// 条件类型
Condition::matches("field", value)           // 精确匹配
Condition::range("field", Range { gte, lte }) // 范围查询
Condition::is_empty("field")                  // 空值检查
Condition::has_id([...])                      // ID过滤
```

## 3. 架构特点

### 3.1 存储架构

```
┌─────────────────────────────────────────────────────────────┐
│                      Qdrant Server                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │   gRPC API  │  │  REST API   │  │  Web UI     │         │
│  │  (Port 6334)│  │ (Port 6333) │  │ (Port 6333) │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
├─────────────────────────────────────────────────────────────┤
│                    Collection Manager                        │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  Segment 1  │  Segment 2  │  Segment 3  │  ...      │   │
│  └─────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │  HNSW Index │  │ Payload Idx │  │    WAL      │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
├─────────────────────────────────────────────────────────────┤
│                    Storage Layer                             │
│  ┌─────────────┐  ┌─────────────┐                          │
│  │   Memory    │  │    Disk     │                          │
│  └─────────────┘  └─────────────┘                          │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 多向量支持

Qdrant支持在一个Collection中存储多种类型的向量：

```json
{
  "vectors": {
    "image": { "size": 512, "distance": "Dot" },
    "text": { "size": 768, "distance": "Cosine" }
  }
}
```

### 3.3 稀疏向量支持

```json
{
  "sparse_vectors": {
    "text-sparse": {}
  }
}
```

## 5. 与GraphDB集成价值

### 5.1 功能增强

| 功能 | 描述 |
|------|------|
| **语义搜索** | 基于向量相似度的语义搜索，超越关键词匹配 |
| **混合检索** | 图遍历 + 向量相似度的混合检索 |
| **智能推荐** | 基于图结构和向量相似度的推荐 |
| **RAG支持** | 支持检索增强生成应用场景 |

### 5.2 架构一致性

| 对比项 | 全文检索 (BM25/Inversearch) | 向量检索 (Qdrant) |
|--------|---------------------------|------------------|
| 存储引擎 | Tantivy/自研 | Qdrant |
| 索引类型 | 倒排索引 | HNSW |
| 查询方式 | 关键词匹配 | 向量相似度 |
| 同步机制 | SyncManager | SyncManager (复用) |
| 协调器 | FulltextCoordinator | VectorCoordinator |

### 5.3 技术栈统一

- **语言**: Rust (与GraphDB一致)
- **异步运行时**: Tokio
- **通信协议**: gRPC
- **配置格式**: TOML/YAML
